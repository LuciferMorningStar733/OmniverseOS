# OMNIVERSEOS TASK — MAKE THE MOBILE EXPERIENCE FEEL LIKE A REAL ANDROID OS

## PROJECT CONTEXT

Project: OmniverseOS

Current status:
The website is now responsive and usable on mobile, but it still feels like a desktop website scaled down instead of a premium Android operating system.

Do NOT redesign the branding or desktop UI.

The desktop experience is already close to perfect.

The goal is to make the mobile experience feel intentionally designed for Android while preserving the exact desktop experience.

Do not modify desktop layout.

# MOBILE EXPERIENCE

Instead of shrinking desktop windows...

Transform them into native app experiences.

When opening an application:

• Animate from dock icon
• Expand into fullscreen
• Respect Android safe areas
• Rounded corners
• Smooth spring animation

No tiny floating desktop windows.

No dragging.

No resizing.

Apps should feel installed.

# DOCK

Current dock feels like desktop.

Improve it.

Requirements:

• Large 56-64px touch targets
• Better spacing
• Floating glass dock
• Blur background
• Safe area padding
• Thumb friendly
• Slight bounce animation on tap
• Active app glow
• Horizontal scrolling if too many apps

# SEARCH

• Floating search pill
